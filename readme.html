🤖 Agetic SDLC Companion (Multi-Agent)

An intelligent, modular system powered by Streamlit and Gemini for transforming raw requirements into fully formed designs, backend code, review documents, test cases, and deployment strategies — with interactive feedback loops and visual diagrams.

🚀 Features

  • ✨ Modular agent pipeline:

    • 🧮 Estimator Agent — Identifies the total effort required for the work
    • 📋 Analyst Agent — Spec generation from raw input
    • 🧱 Design Agent — Architecture + Diagram (via Gemini Vision)
    • 💻 Backend Agent — Code generation
    • 🧐 Review Agent — Code critique
    • 🧪 Test Agent — Validation logic
    • 🚀 Deploy Agent — Strategy planning

  • 💬 Editable user feedback per phase

  • 🖼️ Dynamic system diagrams using Gemini Vision

  • 📥 One-click download for each agent’s output

  • 🎨 Branded UI with banner, icons, and light-mode config
